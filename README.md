# Flujo-de-control
Crear un programa en C# que lea el sueldo de un empleado, calcule el ISR (Impuesto 
Sobre la Renta) y muestre en pantalla el resultado, de acuerdo con una escala salarial 
disponible en la plataforma de la DGII.  Para realizar el cálculo, debe investigar cómo se 
realiza para este país y en base a este año.  El resultado debe ser el sueldo y cuanto 
debe pagar por el ISR, si no aplica, colocar N/A.
